THE INFLUENCE
Introduction

Truly good educational materials may be repetitive or even boring, but they must always be easy for children to understand.

Our materials follow this principle: while they may sometimes feel repetitive, they are carefully designed to be clear and accessible for young learners.

We believe that if an average 10-year-old can easily understand an idea, then an average adult will too.

The Mind

We want to train a machine mind to multiply any number by 5.00.

This mind will be called the Multiplier By Five.

It will have a single PARAMETER: WEIGHT.

The mind will be just a formula like that:

OUTPUT = INPUT × WEIGHT
Initial PARAMETER

Suppose we don't yet know what the PARAMETER should be, so let's start by setting it to zero:

WEIGHT = 0.00

At first, the mind will generate nonsense.

The Dataset

We have the training DATASET consisting of two EXAMPLES:

INPUT TARGET
EXAMPLE 1  0.60  3.00
EXAMPLE 2 -1.00 -5.00

EXAMPLE 1 means when the INPUT is 0.60, the OUTPUT must be 3.00.
EXAMPLE 2 means when the INPUT is -1.00, the OUTPUT must be -5.00.

During training, at every LESSON, the mind will update the PARAMETER so that the OUTPUT gets closer to the TARGET.

The Update Rule

The mind updates the PARAMETER using this UPDATE RULE:

Refined Update Rule
PARAMETER = PARAMETER + PARAMETER_update
PARAMETER_update = ERROR × PARAMETER_OUTPUT_influence

About the PARAMETER_OUTPUT_influence

To understand this simple concept, let's look at two examples, each with a simple formula.

1
Example 1

The simple formula:

OUTPUT = 3.0 × PARAMETER

If PARAMETER = 5.0, then OUTPUT = 3.0 × 5.0 = 15.0.

Now, if we change the PARAMETER by +0.1, the OUTPUT changes by +0.3:

PARAMETER = 5.0 + 0.1 = 5.1
OUTPUT = 3.0 × 5.1 = 15.3

PARAMETER_change = 5.1 − 5.0 = 0.1
OUTPUT_change = 15.3 − 15.0 = 0.3

MULTIPLIER = OUTPUT_change / PARAMETER_change = 0.3 / 0.1 = 3.0

👉 The OUTPUT changed 3.0 times as much as the PARAMETER.

So we say: The INFLUENCE of the PARAMETER on the OUTPUT is 3.0:

PARAMETER_OUTPUT_influence = 3.0

Takeaway
The number (3.0) that multiplies the PARAMETER is the INFLUENCE of the PARAMETER on the OUTPUT.

OUTPUT = 3.0 × PARAMETER
PARAMETER_OUTPUT_influence = 3.0
2
Example 2

The simple formula:

OUTPUT = −3.0 × PARAMETER

If PARAMETER = 5.0, then OUTPUT = −3.0 × 5.0 = −15.0.

Now, if we change the PARAMETER by +0.1, the OUTPUT changes by −0.3:

PARAMETER = 5.0 + 0.1 = 5.1
OUTPUT = −3.0 × 5.1 = −15.3

PARAMETER_change = 5.1 − 5.0 = 0.1
OUTPUT_change = −15.3 − (−15.0) = −0.3

MULTIPLIER = OUTPUT_change / PARAMETER_change = −0.3 / 0.1 = −3.0

👉 The OUTPUT changed 3.0 times as much as the PARAMETER, but in the opposite direction.

So we say: The INFLUENCE of the PARAMETER on the OUTPUT is −3.0.

PARAMETER_OUTPUT_influence = -3.0

Takeaway
The number (-3.0) that multiplies the PARAMETER is the INFLUENCE of the PARAMETER on the OUTPUT.

OUTPUT = −3.0 × PARAMETER
PARAMETER_OUTPUT_influence = −3.0
The universal takeaway for simple formulas:
Universal Takeaway
The number that multiplies a PARAMETER is always the INFLUENCE of that PARAMETER on the OUTPUT.

Now we can write the refined rule again, this time with full understanding:

Refined Update Rule (restated)
PARAMETER = PARAMETER + PARAMETER_update
PARAMETER_update = ERROR × PARAMETER_OUTPUT_influence

Because the mind has just one PARAMETER — WEIGHT — we obtain:

Refined Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR × WEIGHT_OUTPUT_influence

Don't worry — it's really simple. Just follow the calculations below.

Training
LESSON 1
WEIGHT = 0.00

EXAMPLE 1:

INPUT = 0.60
TARGET = 3.00
Forward Pass
PRODUCT = INPUT × WEIGHT = 0.60 × 0.00 = 0.00
OUTPUT = PRODUCT = 0.00
Error
ERROR = TARGET − OUTPUT = 3.00 − 0.00 = 3.00
Backward Pass — Backpropagation
Refined Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR × WEIGHT_OUTPUT_influence
WEIGHT = 0.00
ERROR = 3.00
What is WEIGHT_OUTPUT_influence here?
Our formula is: OUTPUT = INPUT × WEIGHT
The number that multiplies the WEIGHT is the INPUT.
Therefore: WEIGHT_OUTPUT_influence = INPUT
WEIGHT_OUTPUT_influence = INPUT = 0.60
WEIGHT_update = ERROR × WEIGHT_OUTPUT_influence = 3.00 × 0.60 = 1.80
WEIGHT = WEIGHT + WEIGHT_update = 0.00 + 1.80 = 1.80

Because the ERROR was 3.00 and the WEIGHT_OUTPUT_influence was 0.60, we updated the WEIGHT by 1.80 (3.00 × 0.60). So the WEIGHT becomes 1.80.
WEIGHT = 1.80
What just happened?

The mind has just improved itself.

Now, if it receives the same INPUT of 0.60, the OUTPUT becomes 1.08 — which is closer to the TARGET (3.00) than the previous OUTPUT (0.00) was.

The ERROR becomes smaller:

ERROR = 1.92


LESSON 1 ends here.

LESSON 2
WEIGHT = 1.80

EXAMPLE 2:

INPUT = -1.00
TARGET = -5.00
Forward Pass
PRODUCT = INPUT × WEIGHT = -1.00 × 1.80 = -1.80
OUTPUT = PRODUCT = -1.80
Error
ERROR = TARGET − OUTPUT = -5.00 − (−1.80) = -3.20
Backward Pass — Backpropagation
Refined Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR × WEIGHT_OUTPUT_influence
WEIGHT = 1.80
ERROR = -3.2
What is WEIGHT_OUTPUT_influence here?
Our formula is: OUTPUT = INPUT × WEIGHT
The number that multiplies the WEIGHT is INPUT.
Therefore: WEIGHT_OUTPUT_influence = INPUT
WEIGHT_OUTPUT_influence = INPUT = −1.00
WEIGHT_update = ERROR × WEIGHT_OUTPUT_influence = −3.20 × (−1.00) = 3.20
WEIGHT = WEIGHT + WEIGHT_update = 1.80 + 3.20 = 5.00

Because the ERROR was −3.20 and the WEIGHT_OUTPUT_influence was −1.00, we updated the WEIGHT by 3.20 (−3.20 × −1.00). So the WEIGHT becomes 5.00.
WEIGHT = 5.00
What just happened?

The mind has just improved itself.

Now, if it receives the same INPUT of −1.00, the OUTPUT becomes −5.00 — which is closer to the TARGET (−5.00) than the previous OUTPUT (−1.80) was.

Wait — it's not just closer. It's exact!

The OUTPUT is now -5.00 — exactly the same as the TARGET.

The ERROR becomes zero:

ERROR = 0.00


LESSON 2 ends here.

✓   DONE   ✓
Testing

To test the mind, we give it a problem — an EXAMPLE — it has never encountered before.

WEIGHT = 5.00

EXAMPLE FOR TESTING:

INPUT = 10.00
TARGET = 50.00
Forward Pass
PRODUCT = INPUT × WEIGHT = 10.00 × 5.00 = 50.0
OUTPUT = PRODUCT = 50.00
Error
ERROR = TARGET − OUTPUT = 50.00 − 50.00 = 0.00

This mind — since ERROR = 0.00 — works perfectly!

It has learned the rule: multiply any number by 5.0.

The Multiplier By Five can now solve problems it has never encountered before.

The mind is now just a formula:

OUTPUT = INPUT × 5.00
