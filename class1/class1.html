Our Very First Machine Mind
Introduction

The words "learning" and "training" are used interchangeably.

The Mind

We want to TRAIN a machine mind that may LEARN to multiply any number by 5.0.

The mind will have a single PARAMETER: WEIGHT.

Initial PARAMETER

Suppose we don't yet know what this PARAMETER should be, so let's start by setting it to zero:

WEIGHT = 0.0

At first, the mind will generate nonsense.

The Dataset

We have the following training DATASET:

# EXAMPLE 1 EXAMPLE 2
INPUT 0.6 1.0
TARGET 3.0 5.0

EXAMPLE 1 means when INPUT is 0.6000, the OUTPUT must be 3.0000.
EXAMPLE 2 means when INPUT is -10.0000, the OUTPUT must be -50.0000.

During TRAINING, at every LESSON, the mind will update the PARAMETER to improve itself.

The Update Rule

The mind updates the PARAMETER using this update rule:

Simple Update Rule
PARAMETER = PARAMETER + PARAMETER_update
PARAMETER_update = ERROR

Since the mind has one PARAMETER, which is WEIGHT, we get:

Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR

Don't worry — it's really simple. Just follow the calculations below.

Training
LESSON 1
WEIGHT = 0.0

EXAMPLE 1:

INPUT  = 0.6
TARGET = 3.0
Forward Pass — Inference
PRODUCT = INPUT × WEIGHT = 0.6 × 0.0 = 0.0
OUTPUT  = PRODUCT = 0.0
Error
ERROR   = TARGET − OUTPUT = 3.0 − 0.0 = 3.0
Backward Pass — Backpropagation
Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR
WEIGHT_update = ERROR = 3.0
WEIGHT = WEIGHT + WEIGHT_update = 0.0 + 3.0 = 3.0

Since ERROR was 3.00, we updated WEIGHT by exactly 3.0. So WEIGHT becomes 3.0.
WEIGHT = 3.0
What just happened?

The mind has just improved itself.

Now, if it receives the same INPUT of 0.60, OUTPUT becomes 1.8 — which is closer to TARGET 3.0 than the previous OUTPUT 0.0 was.

ERROR becomes smaller:

ERROR = 1.2


Lesson 1 finishes here.

LESSON 2
WEIGHT = 3.0

EXAMPLE 2:

INPUT  = 1.0
TARGET = 5.0
Forward Pass — Inference
PRODUCT = INPUT × WEIGHT = 1.0 × 3.0 = 3.0
OUTPUT  = PRODUCT = 3.0
Error
ERROR   = TARGET − OUTPUT = 5.0 − 3.0 = 2.0
Backward Pass — Backpropagation
Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR
WEIGHT_update = ERROR = 2.0
WEIGHT = WEIGHT + WEIGHT_update = 3.0 + 2.0 = 5.0

Since ERROR was 2.0, we updated WEIGHT by exactly 2.0. So WEIGHT becomes 5.0.
WEIGHT = 5.0
What just happened?

The mind has just improved itself. Again!

Now, if it receives the same INPUT of 1.0, OUTPUT becomes 5.0 — which is closer to TARGET 5.0 than the previous OUTPUT 3.0 was.

Wait — it's not just closer. It's exact!
OUTPUT is now 5.0 — exactly the same as TARGET.

ERROR becomes zero:

  ERROR = 0.0


Lesson 2 finishes here.

✓   DONE   ✓
Testing

To test the mind, we give it a problem — an EXAMPLE — it has never encountered before.

WEIGHT = 5.0

EXAMPLE FOR TESTING :

INPUT  = 10.0
TARGET = 50.0
Forward Pass
PRODUCT = INPUT × WEIGHT = 10.0 × 5.0 = 50.0
OUTPUT  = PRODUCT = 50.0
ERROR   = TARGET − OUTPUT = 50.0 − 50.0 = 0.0

This mind — since ERROR = 0.0 — works perfectly!

It has learned the rule: multiply any number by 5.0.

The Multiplier By Five can solve problems it has never encountered before.
