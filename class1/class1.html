Class 1
The Multiplier by Five
Introduction

The words "learning" and "training" are used interchangeably.

The Mind We Want to Build

We want to create a machine mind that can multiply any number by 5.00.

This mind will consist of a single PARAMETER: WEIGHT.

Suppose we don't know what value this WEIGHT should have, so we initially set it to zero:

WEIGHT = 0.00

At first, the mind will generate nonsense.

Dataset

We have the following training dataset:

# INPUT TARGET
SAMPLE 1 0.60 3.00
SAMPLE 2 1.00 5.00

SAMPLE 1 means when INPUT is 0.60, the OUTPUT must be 3.00.
SAMPLE 2 means when INPUT is 1.00, the OUTPUT must be 5.00.

When training, at every lesson, we will update the PARAMETER (WEIGHT) to improve the mind.

The Update Rule

We will update the PARAMETER using this update rule:

Simple Update Rule
PARAMETER = PARAMETER + PARAMETER_UPDATE
PARAMETER_UPDATE = ERROR

Since we have just one PARAMETER, which is WEIGHT, we obtain:

Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_UPDATE
WEIGHT_UPDATE = ERROR

Don't worry — it's really simple. Just follow the calculations below.

Training
LESSON 1
WEIGHT = 0.00
SAMPLE 1:
INPUT    = 0.60
TARGET = 3.00
Forward Pass
PRODUCT = INPUT × WEIGHT = 0.60 × 0.00 = 0.00
OUTPUT  = PRODUCT = 0.00
ERROR   = TARGET − OUTPUT = 3.00 − 0.00 = 3.00
Backward Pass — Backpropagation
Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_UPDATE
WEIGHT_UPDATE = ERROR
WEIGHT = 0.00
ERROR = 3.00
WEIGHT_UPDATE = ERROR = 3.00
WEIGHT = WEIGHT + WEIGHT_UPDATE = 0.00 + 3.00 = 3.00

Since the ERROR was 3.00, we adjusted the WEIGHT by exactly 3.00. So the WEIGHT becomes 3.00.
WEIGHT = 3.00
What just happened?

We have just improved the mind.

Now, if the mind receives the same INPUT of 0.60, the OUTPUT becomes 1.80 — which is closer to the TARGET 3.00 than the previous OUTPUT 0.00 was.

The ERROR becomes smaller:

ERROR = 1.20

Lesson 1 finishes here.

LESSON 2
WEIGHT = 3.00
SAMPLE 2:
INPUT    = 1.00
TARGET = 5.00
Forward Pass
PRODUCT = INPUT × WEIGHT = 1.00 × 3.00 = 3.00
OUTPUT  = PRODUCT = 3.00
ERROR   = TARGET − OUTPUT = 5.00 − 3.00 = 2.00
Backward Pass — Backpropagation
Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_UPDATE
WEIGHT_UPDATE = ERROR
WEIGHT = 3.00
ERROR = 2.00
WEIGHT_UPDATE = ERROR = 2.00
WEIGHT = WEIGHT + WEIGHT_UPDATE = 3.00 + 2.00 = 5.00

Since the ERROR was 2.00, we adjusted the WEIGHT by exactly 2.00. So the WEIGHT becomes 5.00.
WEIGHT = 5.00
What just happened?

We have just improved the mind.

Now, if the mind receives the same INPUT of 1.00, the OUTPUT becomes 5.00 — which is closer to the TARGET 5.00 than the previous OUTPUT 3.00 was.

Wait — it's not just closer. It's exact!
OUTPUT is now 5.00 — exactly the same as TARGET.

The ERROR becomes zero:

  ERROR = 0.00

Lesson 2 finishes here.

✓   DONE   ✓
Validation
WEIGHT = 5.00
VALIDATION SAMPLE:
INPUT    = 10.00
TARGET = 50.00
Forward Pass
PRODUCT = INPUT × WEIGHT = 10.00 × 5.00 = 50.00
OUTPUT  = PRODUCT = 50.00
ERROR   = TARGET − OUTPUT = 50.00 − 50.00 = 0.00

Since ERROR = 0.00, this mind works perfectly (as expected)!

Of course, it will work correctly if we give it any number.

The mind has learned the rule: multiply any number by 5.00.

So, our job here is done.

Conclusion

A mind can learn from its own mistakes (errors), and the UPDATE RULE makes this learning possible. Here, we used the SIMPLE UPDATE RULE, which was sufficient for the given task.
