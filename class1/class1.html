Class 1
Multiplier by Five
Introduction

The words "learning" and "training" are used interchangeably.

The Mind We Want to Grow

We want to grow a mind that can multiply any number by 5.00.

This mind will have a single PARAMETER: WEIGHT.

Initial PARAMETER

Suppose we don't yet know what the WEIGHT should be, so let's start by setting it to zero:

WEIGHT = 0.00

At first, the mind will generate nonsense.

Dataset

We have the following training dataset:

# INPUT TARGET
EXAMPLE 1 0.60 3.00
EXAMPLE 2 1.00 5.00

EXAMPLE 1 means when INPUT is 0.60, the OUTPUT must be 3.00.
EXAMPLE 2 means when INPUT is 1.00, the OUTPUT must be 5.00.

When training, at every lesson, the mind will update the PARAMETER to improve itself.

The Update Rule

The mind will update the PARAMETER using this update rule:

Simple Update Rule
PARAMETER = PARAMETER + PARAMETER_update
PARAMETER_update = ERROR

Since the mind has just one PARAMETER, which is WEIGHT, it obtains:

Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR

Don't worry — it's really simple. Just follow the calculations below.

Training
LESSON 1
WEIGHT = 0.00

EXAMPLE 1:

INPUT  = 0.60
TARGET = 3.00
Forward Pass — Inference
PRODUCT = INPUT × WEIGHT = 0.60 × 0.00 = 0.00
OUTPUT  = PRODUCT = 0.00
Error
ERROR   = TARGET − OUTPUT = 3.00 − 0.00 = 3.00
Backward Pass — Backpropagation
Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR
WEIGHT = 0.00
ERROR = 3.00
WEIGHT_update = ERROR = 3.00
WEIGHT = WEIGHT + WEIGHT_update = 0.00 + 3.00 = 3.00

Since the ERROR was 3.00, we adjusted the WEIGHT by exactly 3.00. So the WEIGHT becomes 3.00.
WEIGHT = 3.00
What just happened?

The mind has just improved itself.

Now, if it receives the same INPUT of 0.60, the OUTPUT becomes 1.80 — which is closer to the TARGET 3.00 than the previous OUTPUT 0.00 was.

The ERROR becomes smaller:

ERROR = 1.20


Lesson 1 finishes here.

LESSON 2
WEIGHT = 3.00

EXAMPLE 2:

INPUT  = 1.00
TARGET = 5.00
Forward Pass — Inference
PRODUCT = INPUT × WEIGHT = 1.00 × 3.00 = 3.00
OUTPUT  = PRODUCT = 3.00
Error
ERROR   = TARGET − OUTPUT = 5.00 − 3.00 = 2.00
Backward Pass — Backpropagation
Simple Update Rule for WEIGHT
WEIGHT = WEIGHT + WEIGHT_update
WEIGHT_update = ERROR
WEIGHT = 3.00
ERROR = 2.00
WEIGHT_update = ERROR = 2.00
WEIGHT = WEIGHT + WEIGHT_update = 3.00 + 2.00 = 5.00

Since the ERROR was 2.00, we adjusted the WEIGHT by exactly 2.00. So the WEIGHT becomes 5.00.
WEIGHT = 5.00
What just happened?

The mind just improved itself. Again!

Now, if it receives the same INPUT of 1.00, the OUTPUT becomes 5.00 — which is closer to the TARGET 5.00 than the previous OUTPUT 3.00 was.

Wait — it's not just closer. It's exact!
OUTPUT is now 5.00 — exactly the same as TARGET.

The ERROR becomes zero:

  ERROR = 0.00


Lesson 2 finishes here.

✓   DONE   ✓
Testing

To test the mind, we give it a problem — an INPUT — it has never encountered before.

WEIGHT = 5.00

EXAMPLE FOR TESTING :

INPUT  = 10.00
TARGET = 50.00
Forward Pass
PRODUCT = INPUT × WEIGHT = 10.00 × 5.00 = 50.00
OUTPUT  = PRODUCT = 50.00
ERROR   = TARGET − OUTPUT = 50.00 − 50.00 = 0.00

This mind — since ERROR = 0.00 — works perfectly!

It has learned the rule: multiply any number by 5.00.

The Multiplier By Five can solve problems it has never encountered before.
